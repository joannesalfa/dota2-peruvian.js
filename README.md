# dota2-peruvian.js
for trolling purposes


#1) Call peruvian traits:
```
peruvianFaceNum();
```

#*: you can choose 1,2,3,4,5 numbers.
For example

```
peruvianFaceNum(3);
```

Outputs: "Small eyes"


#2) Call random peruvian traits:
```
peruvianFaceRandom();
```

It returns one of the 5 peruvian traits in random.
